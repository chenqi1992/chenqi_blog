# TypeScript 基本类型

TypeScript 提供了丰富的类型系统。

## 常见基本类型
- number
- string
- boolean
- array
- tuple
- enum
- any
- void
- null/undefined

## 示例

```ts
let age: number = 18;
let name: string = '张三';
let isOk: boolean = true;
let arr: number[] = [1, 2, 3];
```

## 小结
掌握基本类型是学习 TypeScript 的第一步。
